I build scientific Python tools for computational chemistry, molecular thermodynamics, chemical data, and molecular/materials modeling.

[Google Scholar](https://scholar.google.com/citations?user=hiHTZ2MAAAAJ) · [ORCID](https://orcid.org/0000-0002-4452-2025)

My work focuses on turning chemically meaningful models and datasets into reusable software: data-access packages, workflow tooling, geometric infrastructure for periodic structures, and ML-ready representations for chemistry.

**Open-source directions:** [DeloneCommons](https://github.com/DeloneCommons) for computational geometry and periodic-structure tools; [muCommons](https://github.com/muCommons) for molecular thermodynamics, chemical data, QChem workflows, and ML-ready chemistry datasets.

**Areas:** scientific Python · computational chemistry · cheminformatics · molecular thermodynamics · ionic liquids · chemical data infrastructure · crystallography · periodic graphs · Voronoi/Laguerre methods · ML for chemistry
